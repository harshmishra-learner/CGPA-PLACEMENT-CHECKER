# 🎓 CGPA → Placement Eligibility Checker

A free, instant tool for B.Tech / AKTU students to:
- Convert CGPA to percentage using multiple formula options
- Check eligibility for 14+ top companies (TCS, Infosys, Wipro, HCL, Accenture, and more)
- Get personalized next-step tips based on their profile

**Live Demo:** https://cgpa-placement-checker.vercel.app/
**Built by:** Harsh Mishra— mishra30harsh@gmail.com

---

## 🚀 Run Locally (VS Code)

No build step needed — this is plain HTML/CSS/JS.

### Option 1 — Live Server (Recommended)
1. Open folder in VS Code
2. Install the **Live Server** extension (ritwickdey.LiveServer)
3. Right-click `index.html` → **Open with Live Server**
4. Browser opens at `http://127.0.0.1:5500`

### Option 2 — Direct Open
Just double-click `index.html` — works in any browser.

## 📐 Formula Used

| University System | Formula |
|---|---|
| AKTU / Most Indian | `(CGPA − 0.75) × 10` |
| Generic (Anna Univ etc.) | `CGPA × 9.5` |
| Direct × 10 | `CGPA × 10` |
